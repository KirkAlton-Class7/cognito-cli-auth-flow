# Lab Sandbox

This folder is intentionally kept empty as a temporary workspace for lab edits.

Use it to copy starter files, make changes, compare against finished references, or stage code before deployment. Organize it however works best for your workflow.

## Example Structure

These are only examples. Use any folder structure that helps you work through the lab.

```text
sandbox/
├── scripts/
├── lambda-code/
├── config/
└── terraform/
```

| Example path | Possible use |
| --- | --- |
| `sandbox/scripts/` | Edited helper scripts or command-line tools |
| `sandbox/lambda-code/` | Edited function source files |
| `sandbox/config/` | Temporary config, policies, or examples |
| `sandbox/terraform/` | Terraform or IaC lab files |
